# Windows XP Portfolio  

This project is a retro-styled portfolio website inspired by the classic Windows XP operating system. It showcases a unique and nostalgic design while demonstrating modern web development techniques using Web Components, Shadow DOM, and modular JavaScript.  

## Features  

- **Custom Web Components**: Modular and reusable components such as `xp-modal`, `xp-tab`, `xp-startmenu`, and more.  
- **Draggable Windows**: Authentic Windows XP-style draggable and resizable windows.  
- **Interactive Start Menu**: A fully functional start menu with pinned programs and shortcuts.  
- **PDF Viewer**: Integrated PDF viewer for displaying resumes or documents.  
- **Web Quiz**: A fun and interactive web development quiz.  
- **Contact Form**: A form to send messages directly from the portfolio.  
- **Responsive Design**: Optimized for various screen sizes.  

## Technologies Used  

- **HTML/CSS**: For structure and styling, including custom styles for the Windows XP theme.  
- **JavaScript**: For interactivity and custom web components.  
- **Parcel**: A modern web application bundler for development and production builds.  
- **npm**: For managing dependencies and scripts.  

## Installation  

1. Clone the repository:  
    ```bash  
    git clone https://github.com/ossema90/windows-xp-portfolio.git  
    ```  

2. Navigate to the project directory:  
    ```bash  
    cd windows-xp-portfolio  
    ```  

3. Install dependencies:  
    ```bash  
    npm install  
    ```  

4. Start the development server:  
    ```bash  
    npm start  
    ```  

5. Open the project in your browser at `http://localhost:1234`.  

## Build for Production  

To create a production build, run:  
```bash  
npm run build  
```  

The output will be available in the `dist` folder.  

## Deployment  

To deploy the project to GitHub Pages, run:  
```bash  
npm run deploy  
```
